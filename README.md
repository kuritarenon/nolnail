# nolnail — 進級制作の Web サイト

kuritarenon/nolnail は、進級制作として制作された Web サイトのソースコードです。
HTML / CSS / JavaScript を使用した静的 Web サイトです。

---

## 📌 プロジェクト概要

本プロジェクトは、実際のネイルサロン「Nol」さんの Web サイト制作を通して、
デザイン設計・コーディング・レスポンシブ対応などのスキルを学ぶことを目的としています。

---

## 📁 ファイル構成

```
nolnail/
├── css/                 # コンパイル済み CSS
├── images/              # 画像素材
├── js/                  # JavaScript ファイル
├── sass/                # Sass / SCSS ファイル
├── index.html           # トップページ
├── access.html          # アクセスページ
├── menu.html            # メニューページ
├── news.html            # お知らせ一覧
├── news-detail.html     # お知らせ詳細
├── reservation.html    # 予約ページ
└── LICENSE              # ライセンス
```
